# Bank-customer-churn-analysis
## 📊 Project Overview
This Power BI project analyzes customer churn patterns using a dataset of 10,000 bank customers from Kaggle.  
The dashboard highlights key drivers of churn, customer demographics, and retention strategies.

## 🛠️ Tools & Technologies
- Power BI (data visualization, DAX measures)
- Kaggle dataset

## 📈 Key Insights
- Identified high‑risk customer segments based on age, tenure, and product usage.
- Visualized churn trends across geography and customer categories.
- Provided actionable recommendations for retention.
